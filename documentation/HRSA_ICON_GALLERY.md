## How to Access Actual Icons

### Method 1: Open in Visio
```
1. Open Microsoft Visio
2. File > Open > HRSA Visio 508 Compliant Template V1.0.vsdx
3. View stencils panel on left
4. Drag icons onto page
```

### Method 2: Export as Images
```powershell
# Run the export script
cd "G:\My Drive\06_Skills\_tools"
python export_visio_icons_as_images.py
```

This will export all 100 icons as PNG images to:
`G:\My Drive\06_Skills\documentation\hrsa_icons\`

## Section 508 Compliance

**When using these icons:**
- ✅ Always include text labels
- ✅ Use Section 508 color palette
- ✅ Ensure 4.5:1 contrast ratio minimum
- ✅ Don't rely on color alone for meaning

**See:** [skill_section_508_color_palette](skill_section_508_color_palette.md)
